# Python 课程体系 【三个阶段】
## 总体教学目标
- 熟练掌握Python语言；
- 具备看懂大部分Python代码的能力；
- 初步具备参加Python相关的编程竞赛的能力；
- 能够使用Python为日常生活提升效率；
- 能够使用Python从网络自动的获取所需要的信息和数据；
- 能够使用Python解决一部分数学问题，例如算法和几何。对函数具有更深的理解，初步掌握对数据和曲线进行拟合的方法；
- 初步了解关系型数据结构；
- 能够主动利用一些开源项目为自己服务；
